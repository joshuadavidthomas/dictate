# Installing bar-rs on Arch Linux
## From pre-built package:
```sh
wget https://crabbar.org/dist/bar-rs/arch/bar-rs-git-0.1.0.41.3beb386-1-x86_64.pkg.tar.zst
sudo pacman -U bar-rs-git-0.1.0.41.3beb386-1-x86_64.pkg.tar.zst
```
